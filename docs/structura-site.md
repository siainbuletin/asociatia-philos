# Structura site-ului

Site-ul este structurat în mai multe părți.

Componente:

- arhiva de documente oficiale
  - conține toate versiunile documentelor oficiale care trec prin Adunarea Generală și devin publice pentru toată lumea
- comunicate și anunțuri
- conținutul paginii de instagram a asociației, care este publicat și pe acest site

Ar trebui să existe și o opțiune să vezi task tracker-ul fiecărei persoane incluse în Biroul de Conducere/Consiliul Director, însă numai pentru persoanele autorizate.

În ceea ce privește structura vizuală (pentru persoanele din departamentulPR), totul este pus într-un folder numit "assets".
